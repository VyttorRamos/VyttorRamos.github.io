## 🛠️ Tecnologias Utilizadas no Portfólio
Este site foi construído de forma responsiva e sem uso de frameworks pesados, focando em performance e código limpo:
* **HTML5** (Semântica e estruturação)
* **CSS3** (Estilização, Flexbox/Grid, Dark/Light Mode e animações)
* **JavaScript** (Vanilla JS para manipulação de DOM e tema)
* **Font Awesome** (Ícones visuais)
